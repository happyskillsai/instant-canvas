# Gotchas

Lessons learned the hard way so we don't repeat them.

- [Runtime (kernel & CLI)](gotchas/runtime.md) — stdout flush truncation, keep-alive poll races, same-version kernel staleness, health-ping liveness, `/tmp` symlinks, re-serializing a file to add one field, a validator error becoming a wall of red in the reader's browser, safe collection deletion.
- [Frontend](gotchas/frontend.md) — CSP dropping inline styles (Plotly's colorbar, markdown-it's table alignment), markdown-it silently refusing `data:image/svg+xml`, why Shiki cannot be used and highlight.js can, Plotly's `<style>` injection and its escape hatch, an `<img>`-linked SVG that cannot be themed, WebGL contexts that are never released, a chart vanishing with no error, `splom` drawing nothing on two dimensions, opaque fills, the `options` merge contract, clicks that detach the element they landed on, string-built `<mark>` highlighting, keyboard-opened modals with nowhere to restore focus, a body scroll lock that locks nothing, hover-revealed controls that a touch screen cannot reach, themeRiver dates, swiftshader printing blank 3D charts that look fine on screen, the sheet-height invariant behind document-mode pagination.
- [Testing](gotchas/testing.md) — a green suite that proves nothing, virtual time hiding races, Chrome's `Host`-header trap, clicking an element before its handler is bound, a throwing hook that blames the wrong test, tests that cannot fail, Node 24 subtest socket isolation, in-runner servers invisible to subprocesses (fake kernels must be child processes), a faked `process.platform` splitting workspace identity, shared state-dir coordination.
- [Packaging (npm CLI & HappySkills skill)](gotchas/packaging.md) — everything in the skill folder ships (why the runtime moved to npm), the npm `files` allowlist and `npm pack --dry-run`, the size caps solved by relocation rather than shrinking, npx tarball testing (`-p` or it executes the file), a 404 on `npm view` not meaning a name is publishable (the typosquat rule that forced the `@happyskillsai` scope), why the vendored Plotly build is not interchangeable, description validators, mandatory `init` scaffolding, the `.claude/` mirror path.
